# Visual Identity

## Status
**Pending extraction from v1.** Next session will extract design tokens from the live v1 app at github.com/ggourliegh/careerkete and populate this file.

## Goals
- V2 must look visually identical to v1 for user continuity
- Extract all CSS variables, typography, component styles
- Document the design system properly
- Build reusable components that match v1's look

## To be extracted
- Colour palette (cream, peach, terracotta, sage, etc.)
- Typography (Fraunces for display, Nunito for body)
- Component styles (buttons, cards, chat bubbles, tabs, forms)
- Spacing scale
- Border radius scale
- Shadow scale
- Animation timings
- Background gradients

## Principle
Users of v1 should not notice any visual difference when using v2, except for the login screen at the start.
